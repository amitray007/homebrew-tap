# homebrew-tap

Public Homebrew tap for [@amitray007](https://github.com/amitray007)'s personal apps.

## Install

```sh
brew tap amitray007/tap
brew install --cask orpheus
```

## Upgrade

```sh
brew upgrade --cask orpheus
```

## Apps

| Cask      | What it is                        |
| --------- | --------------------------------- |
| `orpheus` | Mac IDE built around Claude Code  |

## How the tap is updated

Cask files and release binaries are generated and pushed here automatically by each app's release pipeline. Don't hand-edit cask files — the next release will overwrite them.
